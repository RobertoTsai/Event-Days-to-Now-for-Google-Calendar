# Event Days to Now for Google Calendar

Enhance your Google Calendar experience with Date Prefixer! This extension adds compact time indicators to your events, making it easier to see how close each appointment is at a glance.

Current version: **1.3**

## Install
Chrome Web Store: 
https://chromewebstore.google.com/detail/hceoajkodhbjkoiopgehogkfplgpmpdj

## Key Features:

1. **See event timing at a glance**: Adds compact prefixes to Google Calendar events, such as 5h, 3d, or 1y5d.
2. **Works with future and past events**: Quickly see how much time remains or how many days have passed.
3. **Fine-grained display filters**: Choose which event types receive countdown labels, whether declined events keep labels, and whether past-event labels remain visible.
4. **Localized settings popup**: Supports 13 popup languages with browser-language auto detection and a manual language selector.
5. **Built for Google Calendar**: Supports all-day, timed, birthday, and Google Tasks entries with clean CSS-based labels and automatic updates.
6. **Simple setup**: Open the extension popup to adjust settings and preview long-period display examples.
7. **Customizable labels**: Toggle year-based long-period display and customize year, day, and hour labels, such as 1年5天.
